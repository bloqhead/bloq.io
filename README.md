# [bloq.io](https://bloq.io)
This is the source for [my website](1). It is built in [VueJS](2) and [Nuxt](3), and hosting on [Zeit Now](4).

## Running

* Run locally: `npm run dev`
* Build for production: `npm run build`
* Start: `npm run start`
